# Basic-Incident-Management (ServiceNow)
ServiceNow basic incident management project with automation

## Description
This project demonstrates a basic Incident Management system built using ServiceNow.

## Features
- Custom Field (Issue Type)
- Client Script (Alert for high priority)
- UI Policy (Description mandatory)
- Business Rule (Auto assignment)
- Data Policy (Short description mandatory)

## Workflow
User creates incident → System validates → Auto assigns → Ensures data integrity
